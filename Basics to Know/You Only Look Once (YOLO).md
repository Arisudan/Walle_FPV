# 🎯 YOLO (You Only Look Once) Models

> **YOLO is a family of real-time object detection models that detect and classify objects in a single forward pass.**

---

## 📈 YOLO Evolution

| Version | Released | Developer | Highlights |
|---------|----------|-----------|------------|
| **YOLOv1** | 2016 | Joseph Redmon | First real-time object detector |
| **YOLOv2 (YOLO9000)** | 2017 | Joseph Redmon | Better accuracy, Anchor Boxes |
| **YOLOv3** | 2018 | Joseph Redmon | Multi-scale detection, Darknet-53 |
| **YOLOv4** | 2020 | Alexey Bochkovskiy | CSPDarknet53, Mosaic Augmentation |
| **YOLOv5** | 2020 | Ultralytics | PyTorch implementation, Easy deployment |
| **YOLOX** | 2021 | Megvii | Anchor-free detection |
| **YOLOv6** | 2022 | Meituan | Industrial deployment optimized |
| **YOLOv7** | 2022 | Wang et al. | State-of-the-art real-time detector |
| **YOLOv8** | 2023 | Ultralytics | Anchor-free, Multi-task support |
| **YOLO-NAS** | 2023 | Deci AI | Neural Architecture Search (NAS) optimized |
| **YOLOv9** | 2024 | WongKinYiu et al. | Programmable Gradient Information (PGI), GELAN |
| **YOLOv10** | 2024 | THU-MIG | End-to-End detection, No NMS |
| **YOLO11** | 2024 | Ultralytics | Improved accuracy, efficiency & multi-task support |
| **YOLO12*** | Upcoming | — | Not officially released *(as of now)* |

> 📌 **YOLO11 is currently the latest official model released by Ultralytics.**

# 🏗️ Common YOLO Model Sizes

| Variant | Description |
|----------|-------------|
| **n** | Nano (Fastest, Smallest) |
| **s** | Small |
| **m** | Medium |
| **l** | Large |
| **x** | Extra Large (Highest Accuracy) |

Example:

- YOLO11n
- YOLO11s
- YOLO11m
- YOLO11l
- YOLO11x

## 📌 Key Points

- **YOLO = You Only Look Once**
- Performs **Object Detection** in a single forward pass.
- Designed for **real-time applications**.
- Widely used in **autonomous drones, robotics, surveillance, and autonomous vehicles**.
- Latest official Ultralytics release: **YOLO11**.
